# **App Name**: Run Krishna Run

## Core Features:

- Dynamic Level Selection: Allow users to select between Easy, Normal, and Hard difficulty levels, adjusting game parameters like gravity, lift, gap size, speed, and spawn rate.
- Score Tracking: Display the current score and the high score, persisting the high score using local storage.
- Obstacle Generation: Generate pipes dynamically with varying heights and gaps based on the selected difficulty level.
- Collision Detection: Detect collisions between the player and the pipes, ending the game when a collision occurs.
- Background Animation: Animate the background to simulate movement and create a sense of depth.
- Restart Game: Allow users to restart the game after it ends.

## Style Guidelines:

- Primary color: Vivid cyan (#00FFFF) to evoke a sense of speed and sky.
- Background color: Light cyan (#E0FFFF), desaturated to 20% to match the primary color, but create depth.
- Accent color: Electric green (#7CFC00), analogous to the primary, to draw attention to key elements and interactions.
- Body and headline font: 'Inter' sans-serif font for clear UI and score display.
- Use simple, clean icons for level selection and game controls.
- Maintain a clean and uncluttered layout to ensure focus on the game play area.
- Subtle animations for player movement, pipe generation, and score updates to enhance the user experience.
